# 前言

随着新冠疫情的蔓延，高校防疫工作成为了重中之重。为了提高防疫工作效率，我们基于SSM框架开发了一款高校防疫监控系统。本项目旨在帮助高校实现疫情信息的实时监控、数据分析与处理，为校园防疫工作提供有力支持。

# 内容介绍

本项目主要包含以下功能模块：用户管理、疫情信息管理、健康监测、通知公告、数据分析等。系统采用前后端分离的设计模式，前端使用Vue.js框架，后端采用Spring、SpringMVC、MyBatis框架。通过这套系统，高校可以快速掌握师生健康状况，有效降低疫情传播风险。

# 技术介绍
